# MindVault – Instagram Saved Posts Manager

A premium SaaS landing page for a Chrome extension that organizes Instagram saved posts into a structured, searchable vault.

## Stack

- Next.js 16 with the App Router
- Tailwind CSS 4
- TypeScript
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

## Production Checks

```bash
npm run lint
npm run build
```
